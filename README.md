# OSS-coop
-UI 개선-
• 메인 메뉴 구성
  Menu Scene을 생성하고 메인 메뉴에 세 가지 주요 버튼 배치
  : Start(게임 시작), Help(조작 방법 소개), Exit(게임 종료)
• Start 메뉴
  Main Scene으로 전환되어 게임을 플레이할 수 있도록 구현
• Help 메뉴
  Help Scene으로 전환되어 Text로 게임의 조작법을 알려주도록 설정
  조작법에 대한 코멘트 추가
    -UP ARROW : JUMP
    -UP ARROW(DOUBLE) : DOUBLE JUMP
    -DOWN ARROW : DROP
  Back 버튼으로 메인 메뉴로 이동할 수 있도록 구현
• Exit 메뉴
  게임이 종료되도록 구현

-인게임 개선-
• 플레이어 조작
  기존 마우스 클릭 점프에서 키보드 ↑ 방향키로 점프, ↓ 방향키로 즉시 착지하도록 구현
  기존 점프 가속도 방식(jumpforce)을 제거하고 일관된 점프 속도(jumpvelocity)로 변경
• 플레이어 생명 시스템
  플레이어에게 목숨 3개를 부여하고 좌측 상단 하트 이미지로 UI 표시
  장애물에 닿으면 생명 1 감소 + 1.5초 무적 시간을 부여하고 무적 시간 동안 캐릭터 깜빡    임으로 표현, 데드존에 닿으면 즉사하도록 구현
• 카메라 조정
  점프 시 캐릭터의 배경화면 이탈을 방지하여 카메라 사이즈를 기존 5 → 7로 변경
• 배경 개선
  게임 진행 중에 변화를 주기 위해 배경 이동속도를 게임 진행에 따라 점차 증가하도록 설정
  점프 시 캐릭터의 배경화면 이탈을 방지하여 배경 Y스케일을 기존 1 → 1.6로 변경
• 장애물 및 데드존 구분
  장애물 태그를 기존 ‘Dead’ → ‘Obstacle’로 변경
  플레이어가 장애물에 부딪히면 생명 1이 감소하고 데드존에 부딪히면 즉사하도록 구현
• ESC 기능 추가
  게임 오버 후 ESC 키 입력 시 메인 메뉴로 이동이 가능하도록 구현
• 장애물 시인성 개선
  장애물 색상을 배경과 구분되도록 기존 하늘색 → 빨간색으로 변경
